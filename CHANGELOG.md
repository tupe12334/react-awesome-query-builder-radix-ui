# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-XX

### Added

- Initial release of @react-awesome-query-builder/radix-ui
- Core widgets:
  - RadixButton - Action buttons with primary/danger variants
  - RadixButtonGroup - Button grouping component
  - RadixIcon - Icon support using Radix UI icons
  - RadixConjs - Conjunction (AND/OR) selector with toggle groups
  - RadixFieldSelect - Field selector with nested support
  - RadixValueSources - Value source selector (value/field/func)
- Value widgets:
  - RadixText - Text input
  - RadixTextArea - Multi-line text input
  - RadixNumber - Number input with min/max/step
  - RadixBoolean - Boolean switch toggle
  - RadixSelect - Single select dropdown
  - RadixMultiSelect - Multi-select with checkboxes
  - RadixSlider - Range slider for numbers
  - RadixDate - Date picker
  - RadixTime - Time picker
  - RadixDateTime - Date and time picker
- Full SCSS styling with CSS variables
- Dark mode support via prefers-color-scheme
- TypeScript type definitions
- Example app demonstrating all features
- Comprehensive documentation

### Features

- Built with Radix UI primitives for accessibility
- Fully compatible with react-awesome-query-builder v6.7.0+
- Supports all export formats (SQL, MongoDB, JsonLogic, SpEL)
- Customizable via CSS variables
- Works with React 16.8.4+, 17, 18, and 19
